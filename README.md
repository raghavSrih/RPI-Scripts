# RPI-Scripts

Scripts related to Raspberry Pi


# MeschAP Installation
  1. update and upgrade the system
     ```
      sudo apt-get update
      sudo apt-get upgrade
     ```
  2. Install the MeschAP using the installer given MeschAP/Installer.sh
      ```
      cd RPI-Scripts/MeschAP
      ./Installer.sh
      ```
  3. Restart the Pi
